# Unity Project
To setup the unity project, you need a copy of Ultrakill and Vanity Reprised.
What you're going to want to do is generate a project using Vanity Reprised, and then copy the contents of Scripts, ULTRAKILL Assets and ULTRAKILL Others to
their respective directories in the the Unity project.

The Addressable Build System has been taken from https://github.com/wafflethings/WafflesWeapons/tree/dev/unity/WafflesWeapons.Unity with the original developer's permission
